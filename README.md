# M5 - PDF Generate

## Instalação dos pacotes

prossiga com os passos:

1. Crie seu ambiente virtual:
```bash
python -m venv venv
```

2. Ative seu venv:
```bash
# linux:
source venv/bin/activate

# windows:
.\venv\Scripts\activate
```

3. rode o requirements.txt:
``
pip install -r requirements.txt
```
